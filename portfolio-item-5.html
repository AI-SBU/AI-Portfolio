Student Services

a REST API

The goal of this project is to create a RESTful service for students. There are courses that students can take. The API allows users to enroll students to courses.

The API has two entities as follows: Student and Course. Used an embedded database to store student and course objects. The database used is CrudRepository. The API provides various services such as adding/removing student/ course, assigning student to a course, removing a course from student etc. It has a controller for each entity to handle http requests.

To handle errors, custom error handling was implemented. For example, if a request comes from client side to retrieve a student details, and the student doesn't exists, it is a 404-Resource not found error. The client receives a user friendly message indicating that the student does not exist in the repository.

Manual testing was done to verify the API's functionality. Used Postman to send http requests. This project was done by myself. Used Java and Spring Boot.
