Banking System Simulation

of a day in a bank

The goal of this project is to simulate a banking system. In a bank, there are certain number of tellers available. People go in a bank to withdraw, deposit, print transactions, open an account etc. The customers waits in a queue. Depending on the number of tellers available, this process can take some time.

The program takes in the number of tellers and customers as an input and randomly generates the tasks the customers want to perform. The customers can perform 1 or all 3 tasks, withdraw, deposit and print recent transactions.

Currently, working on making a record system to store the customers information. This helps further analyzing different scenarios.

The project development and testing was done by me. Used C++.
